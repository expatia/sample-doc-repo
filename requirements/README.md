# Requirements

This document provides access to the requirements documentation that specifies the requirements for this platform and the software applications suite.

Requirements are divided up into several categories as follows:

| Requirement&nbsp;Category | Description |
| ------------------------- | ----------- |
| [Capabilities](./capabilities/README.md) | These requirements relate Business Capabilities used in Value Streams to fulfil the strategic goals of the business. |
| [Integrations](./integrations/README.md) | These describe the technical requirements for integration with third party systems and organizations |
| [UI](./ui/README.md) | These describe the user interface requirements that drive the User Interface design. |
| [User Stories](./user-stories/README.md) | These document the actors, motivations and desired behaviours of the system |
