# User Stories

This document describes how we use User Stories and where you can find these. In our platform User Stories are held in GitHub repositories as Issues.

User Stories are organised into Epics. The following Epics describe the high level use cases of our system:

| Epic | Description |
| ---- | ----------- |
| Account Management | Describes the functionality of the Account application |
| Asset Management | Describes the functionality of the Asset Manager application.
| Fleet Management | Describes the functionality of the Fleet Management application |
| Tenant Management | Describes the functionality of the Tenant Manager application |
| User Management | Describes the functionality of the User Manager application. |
| Video Surveillance | Describe the functionality of the Command Console application. |
