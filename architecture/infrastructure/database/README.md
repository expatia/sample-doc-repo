# Databases

The following databases are used to persist application and system data within the cloud.


| Database | Description |
| -------- | ----------- |
| [DynamoDB](./dynamodb.md) | This database is used to store fine-grained permissions, configuration data, audit events, user profile data, user preferences, product definitions, asset management details, fleet management details, subscriptions, billing information, payment details and accessibility options. |
| [Timestream for InfluxDB](./timestream.md) | Time series data such as IoT device telemetry. |


Click on one of the links above for more information.
