# Component Software Architecture

This document describes our Componen Architecture and lists the categories of components built into the platform.


## Component Categories

The categories of components built into the platform are listed in the table below.


| Component&nbsp;Category | Description |
| ----------------------- | ----------- |
| [Server](./server/README.md) | Components deployed as self-contained servers in Amazon EC2 or within Docker containers in Amazon ECS |
| [Serverless](./serverless/README.md) | Components deployed as AWS Lambda functions |
| [User Interface (UI)](./ui/README.md) | The common library of UI components used in one or more software applications in the software application suite. |
