# Product Manager Software Architecture

This document describes the software architecture of the Product Manager software application.

Links to other related documents are provided below. Click on those links to find out more.

## Functionality

The Product Manager application provides users of the application suite with functionality to:

- Create a digital software product
- Edit a digital software product
- Create pricing of a digital software product
- Edit pricing of a digital software product
- Delete a digital software product
- Archive a digital software product
