# Fleet Manager Software Architecture

This document describes the software architecture of the Fleet Manager software application.

Links to other related documents are provided below. Click on those links to find out more.

## Functionality

The Fleet Manager application provides users of the application suite with functionality to:

- Create asset deployment configurations
- Edit asset deployment configurations
- Delete asset deployment configurations
- Archive asset deployment configurations
- List asset deployment configurations
- Manage asset telemetry
- Update asset deployment configuration history
- Remotely control asset functions
- View asset deployment locations
- View asset deployment status
- View asset deployment telemetry
- View asset deployment history
