# Reports Application Software Architecture

This document describes the software architecture of the Reports software application.

Links to other related documents are provided below. Click on those links to find out more.

## Functionality

The Reports application provides users of the application suite with functionality to:

- specify a report
- generate a report
- schedule report generation
- create a report distribution list
- edit a report distribution list
- delete a report distribution list
- archive a report distribution list
- activate report generation
- suspend report generation
- view reports
- archive reports
- configure report retention policies
