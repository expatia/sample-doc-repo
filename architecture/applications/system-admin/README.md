# System Admin Software Architecture

This document describes the software architecture of the System Admin software application.

Links to other related documents are provided below. Click on those links to find out more.

## Functionality

The System Admin application provides users of the application suite with functionality to:

- Configure data retention policies
- Configure telemetry frequency 
- Configure third party integration adapters
- Configure logging
- Configure auditing
- View audit logs
- View access logs
