# Map Application Software Architecture

This document describes the software architecture of the Map software application.

Links to other related documents are provided below. Click on those links to find out more.

## Functionality

The Map application provides users of the application suite with functionality to:

- View the location of entities on a map
- View information about those entities in the map display
- Navigate to other applications to show more detailed information about an entity on the map
- View video feeds from assets located on a map
- Find locations and assets on a map
- Save map display configurations with meaningful names
- Retrieve map display configurations with meaningful names
- Display a map defined with a meaningful display configuration.
