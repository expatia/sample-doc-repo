# User Manager Software Architecture

This document describes the software architecture of the User Manager software application.

Links to other related documents are provided below. Click on those links to find out more.

## Functionality

The User Manager application provides users of the application suite with functionality to:

- Create User profiles
- Edit User profiles
- Delete User profiles
- Archive User profiles
- List User profiles
- Create a role
- Edit a role
- Assign a permission to a role
- Unassign a permission from a role
- Delete a role 
- Assign role to a User
- Unassign role from a User
- List roles
- List permissions
- List permissions for a user
- Suspend User access
- Enable User access
