# Dashboard Application

This document describes the software architecture of the Dashboard software application.

Links to other related documents are provided below. Click on those links to find out more.

## Functionality

The Dashboard application provides users of the application suite with functionality to:

- create a named dashboard
- edit a named dashboard
- add widgets to a dashbaord
- delete widges from a dashboard
- position widgets within a dashboard
- resize widgets within a dashboard
- delete a named dashboard
- archive a named dashboard
- view warnings
- view alerts
- configure alerts
