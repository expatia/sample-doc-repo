## Dashboard Widget Catalog

The following dashboard widgets are provided by the system.

| Dashboard&nbsp;Widget | Description |
| --------------------- | ----------- |
| Battery Health | Displays a battery health gauge |
| Connectivity Status | Displays a connectivity status gauge |
| Solar Charging Efficiency | Displays a gauge showing solar charging efficiency |
| Asset Online Status | Displays a gauge showing the number of assets that are online |
| Total Assets | Displays the total number of assets being monitored. |
| Asset Uptime | Displays the uptime of assets |
| Open Alerts | Displays the number of open alerts |
