# Billing Application Software Architecture

This document describes the software architecture of the Billing software application.

Links to other related documents are provided below. Click on those links to find out more.

## Functionality

The Billing application provides users of the application suite with functionality to:

- enter billing information
- update billing information
- add payment information
- update payment information
- view orders
- view invoices
- view receipts
- view payments
- view refunds
